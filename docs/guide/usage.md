# 使用
运行以下命令来创建一个新项目：
```shell
$ yolk init
```
你会被提示选取一个项目模板:
<img :src="$withBase('/cli-init.png')" alt="yolk-cli">

根据你项目使用的技术栈选择项目模板，选择完项目模板后你会被提示输入项目的名称：
<img :src="$withBase('/cli-input-name.png')" alt="yolk-cli">

输入项目名称后，即会下载你选择的项目模板：
<img :src="$withBase('/cli-download.png')" alt="yolk-cli">
